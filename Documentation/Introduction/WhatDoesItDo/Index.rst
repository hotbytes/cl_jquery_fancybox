What does it do?
^^^^^^^^^^^^^^^^

This extension adds the jQuery lightbox script
“fancyBox” in version 2 or 3 to your TYPO3 website.

Features:

- It comes with many options to setup the FancyBox as you want.

- Compatible with tt\_news

- HTML5 compatible (rel attribute removed)

You can see a demo on my `website <http://www.wapplersystems.de/typo3
-agentur-aachen/t3extensions/fancybox/>`_ .
